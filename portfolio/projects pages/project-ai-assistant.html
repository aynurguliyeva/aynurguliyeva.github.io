Personalized Learning Assistant

An AI assistant that summarizes content and answers questions from PDFs and YouTube videos.

Project Overview

This project is a system designed to improve educational experiences by answering user queries and summarizing course material. Using advanced NLP techniques, including Retrieval-Augmented Generation (RAG) and FAISS vector search, the assistant processes user-uploaded content from PDFs and YouTube transcripts and provides personalized outputs.

My Contributions

  • I contributed to the development of this project with Alfiya Abdimutalipova.
  • We integrated tools like Hugging Face, Google's Gemini LLM, and Groq API.
  • The project includes voice-based interaction and content ingestion from PDFs and YouTube transcripts.
  • The system is divided into key components, including Document Processing, LLM Integration, Vector Database, and User Interface.
