Asian Hornet Nest Detection System

Building an AI solution for wildlife monitoring and conservation.

Project Overview

This project involved developing a wildlife AI system to detect Asian hornets from video streams. The goal was to provide an automated, non-invasive method for monitoring and protecting native ecosystems from this invasive species.

My Role & Contributions

  • Manually labeled over 5,000+ frames of video data to create a high-quality dataset for training.
  • Utilized YOLO (You Only Look Once) to build a robust object detection model.
  • Implemented a tracking system, which is still under continuous optimization to improve performance.
  • Successfully solved hardware and power issues to enable the system to be deployed on embedded systems for edge AI applications, ensuring it could function reliably in remote environments.
